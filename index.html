使い方（超短）

  1. 起動 → 再生を押す（最初はサンプルが鳴ります）
  2. URLで追加：mp3直リンク or YouTube のURLを貼って追加
  3. 端末音源：音源ファイルを選ぶで追加（機内モードでもOK）
  4. 曲を切り替える：一覧のトグルをON → 再生ボタン
GitHub Pagesで公開後、iPhone Safariで開いて「共有 → ホーム画面に追加」でアプリっぽく使えます。
